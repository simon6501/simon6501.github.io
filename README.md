## Install Ruby
https://rubyinstaller.org/downloads/
```
ruby -v
```

## Install RubyGem
```
gem update --system
gem -v
```

## Install GCC and Make
https://code.visualstudio.com/docs/cpp/config-mingw
```
gcc -v
g++ -v
make -v
```

## Install Jekyll
```
gem install jekyll bundler
jekyll -v
```

## Restart the PC

## Install necessary packages
```
bundle install
```

## Run the project
```
bundle exec jekyll serve
```
